## Neuronin input

$$
z= \left ( \sum_{i=0}^{n} w_{i} x_{i} \right ) + b,
$$

jossa

* $x$ koostuu edellisen kerroksen lähettämistä arvoista.
* $b$ on kerroksen "bias"
* $W$ on edellisen kerroksen painot.
